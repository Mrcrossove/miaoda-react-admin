# 小红书搜索功能快速开始

## 功能说明

"帮我选品"功能已接入真实的小红书搜索API，可以搜索真实的爆款笔记数据。

## ✅ 配置状态

**已完成配置！** 小红书Cookie和API Key已成功配置到Supabase环境变量中，可以直接使用。

## 使用方法

1. 打开应用的"帮我选品"页面
2. 输入关键词（如"口红"、"美妆"、"护肤"等）
3. 点击搜索按钮或按回车键
4. 查看真实的小红书爆款笔记列表
5. 点击笔记卡片跳转到小红书查看详情

## 功能特点

- ✅ 搜索真实的小红书笔记数据
- ✅ 自动筛选点赞5000+的爆款内容
- ✅ 按点赞数排序，展示最热门的笔记
- ✅ 显示笔记标题、封面图、作者信息
- ✅ 显示点赞数（如"2万"、"1.5万"等）
- ✅ 响应式布局，支持移动端和桌面端
- ✅ 点击卡片直接跳转到小红书查看详情

## 成本说明

- 每次搜索消耗：5个点数
- 建议每天搜索10-20次
- 根据使用量定期充值

## 注意事项

⚠️ Cookie有效期有限，如果搜索失败请联系管理员更新Cookie
⚠️ 合理使用API，避免频繁搜索相同关键词
⚠️ 建议搜索热门关键词以获得更多结果
