# 自媒体创作应用 - 功能说明

## 核心功能

### 1. 我有产品
**功能描述**：帮助用户为产品生成小红书爆款文案

**使用流程**：
1. 点击"添加产品"按钮
2. 填写产品信息：
   - 产品名称（必填）
   - 产品描述（选填）
   - 核心卖点（选填）
   - 目标用户（选填）
3. 上传产品图片（必填，最多9张）
4. 点击"创建产品"
5. 在产品卡片上点击"生成小红书文案"
6. 系统自动生成文案（流式输出）
7. 文案自动去除特殊符号（*、#等）
8. 点击"复制文案"复制到剪贴板
9. 点击"下载所有图片"下载产品图片
10. 点击"发布到小红书"：
    - 自动复制文案到剪贴板
    - 尝试打开小红书APP
    - 如果APP未安装，打开小红书创作中心网页版
    - 用户手动粘贴文案并上传图片

**技术特点**：
- 使用文心大模型生成小红书风格文案
- 流式输出，实时显示生成过程
- 自动去除文案中的特殊符号
- 支持多张图片上传和批量下载
- 智能跳转小红书APP或网页版

### 2. 帮我选品
**功能描述**：搜索小红书爆款笔记，发现热门产品

**使用流程**：
1. 在搜索框输入赛道名称/行业名称/产品名称
2. 点击"搜索"按钮或按回车键
3. 系统自动搜索点赞5000+的爆款笔记
4. 以卡片形式展示至少10条结果
5. 查看笔记封面、标题、描述、互动数据
6. 点击卡片跳转到小红书查看详情

**展示内容**：
- 笔记封面图（高质量）
- 笔记标题和描述
- 作者信息（头像、昵称）
- 互动数据（点赞、评论、收藏）
- 点赞数标签（显示在封面右上角）

**技术特点**：
- 自动筛选点赞5000+的爆款内容
- 响应式网格布局（移动端1列，平板2列，桌面3列）
- 悬停效果（阴影、图片缩放）
- 数字格式化（超过1万显示为w）
- 点击跳转到小红书查看详情
- 支持回车键快速搜索

## 技术栈

- **前端框架**：React + TypeScript
- **UI组件**：shadcn/ui + Tailwind CSS
- **路由**：React Router v6
- **状态管理**：React Context + Hooks
- **后端服务**：Supabase
- **数据库**：PostgreSQL (Supabase)
- **文件存储**：Supabase Storage
- **AI服务**：文心大模型（小红书文案生成）
- **流式响应**：ky + eventsource-parser

## 注意事项

1. **小红书发布限制**：
   - 小红书没有开放API，无法直接从外部应用发布内容
   - 当前方案是自动复制文案并打开小红书，用户需手动粘贴和上传图片
   - 这是目前最优的用户体验方案

2. **图片上传**：
   - 图片会自动压缩到1MB以下
   - 支持JPEG、PNG、GIF、WEBP、AVIF格式
   - 文件名只能包含英文字母和数字

3. **文案生成**：
   - 使用文心大模型生成
   - 生成时间约5-10秒
   - 自动去除*、#等特殊符号
   - 文案风格符合小红书社区调性

4. **笔记搜索**：
   - ✅ 已配置真实的小红书搜索API
   - ✅ Cookie和API Key已配置完成，可直接使用
   - 每次搜索消耗5个点数
   - 自动筛选点赞5000+的爆款内容
   - 按点赞数排序，返回至少10条结果
   - 支持关键词搜索（赛道/行业/产品）
   - 点击卡片跳转到小红书查看详情

5. **电商拿货平台**：
   - ✅ 支持一键跳转到1688、淘宝、拼多多、京东
   - 输入关键词后自动显示电商平台入口
   - 点击平台按钮在新窗口打开对应搜索结果
   - 提供各平台的使用建议和适用场景
   - 1688适合批发拿货，淘宝适合小批量测试
   - 拼多多价格最优，京东品质保障

## 使用建议

1. **产品信息填写**：
   - 尽量填写完整的产品信息
   - 核心卖点要突出产品优势
   - 目标用户要明确具体

2. **图片选择**：
   - 使用高质量的产品图片
   - 图片要清晰展示产品特点
   - 可以包含使用场景图

3. **文案优化**：
   - 生成后可以根据需要手动调整
   - 复制前检查文案是否符合预期
   - 可以多次生成选择最佳版本

4. **选品搜索**：
   - 使用具体的关键词（如"口红"而不是"化妆品"）
   - 可以搜索品牌名、产品名、品类名
   - 查看爆款笔记的标题、描述、互动数据
   - 分析爆款内容的共同特点
   - 参考爆款笔记的创作思路
