# 小红书JSSDK一键发布功能集成总结

## 功能概述

已成功将应用中的三个核心模块的发布功能统一升级为小红书JSSDK一键发布，实现自动唤起小红书APP并自动填充内容。

## 技术实现

### 核心Hook：useXHSShare

**文件位置**：`src/hooks/useXHSShare.ts`

**主要功能**：
- 检测小红书JS SDK是否加载
- 检测小红书APP是否安装
- 获取鉴权配置（从Edge Function）
- 调用小红书JS SDK分享
- 降级方案（复制文案到剪贴板）

**API接口**：
```typescript
interface XHSShareParams {
  type: 'normal' | 'video';  // 笔记类型
  title?: string;             // 笔记标题
  content?: string;           // 笔记正文
  images?: string[];          // 图片URL数组
  video?: string;             // 视频URL
  cover?: string;             // 视频封面
  fail?: (error: unknown) => void;  // 失败回调
}

const { shareToXhs } = useXHSShare();
await shareToXhs(params);
```

### Edge Function：xhs-auth

**功能**：生成小红书API鉴权配置

**返回数据**：
```typescript
{
  app_key: string;      // 应用唯一标识
  nonce: string;        // 随机字符串
  timestamp: string;    // 时间戳
  signature: string;    // 签名
}
```

### 小红书JS SDK

**官方文档**：https://agora.xiaohongshu.com/doc/js

**核心API**：
```javascript
window.xhs.share({
  shareInfo: {
    type: 'normal',
    title: '标题',
    content: '文案',
    images: ['图片URL1', '图片URL2']
  },
  verifyConfig: {
    appKey: 'xxx',
    nonce: 'xxx',
    timestamp: 'xxx',
    signature: 'xxx'
  },
  fail: (error) => {
    console.error('分享失败:', error);
  }
});
```

## 注意事项

### 必要条件

1. **小红书JS SDK**：
   - 必须在HTML中引入SDK脚本
   - SDK必须成功加载
   - 检查：`window.xhs` 是否存在

2. **小红书APP**：
   - 用户设备必须安装小红书APP
   - APP版本需支持JS SDK调用
   - 检查：使用 `checkXHSInstalled()` 方法

3. **图片URL要求**：
   - 必须是公网HTTPS地址
   - 必须可直接访问
   - 建议使用Supabase Storage

### 参数要求

1. **图文类型（normal）**：
   - title：可选，建议提供
   - content：可选，建议提供
   - images：必填，至少1张图片

2. **视频类型（video）**：
   - title：可选
   - content：可选
   - video：必填，视频URL
   - cover：可选，封面图URL

### 错误处理

1. **SDK未加载**：
   - 自动降级为复制文案
   - 提示用户手动打开小红书

2. **APP未安装**：
   - 提示用户安装小红书
   - 引导下载（打开官网）

3. **发布失败**：
   - 显示错误提示
   - 自动降级为复制文案
   - 调用fail回调函数
