# 自媒体创作应用 - 使用示例

## 功能1：我有产品

### 场景：为口红产品生成小红书文案

**步骤1：添加产品**
- 点击"添加产品"按钮
- 填写产品信息：
  - 产品名称：YSL圆管口红
  - 产品描述：法国奢华品牌，经典圆管设计，丝滑质地
  - 核心卖点：显白、持久、滋润、高级感
  - 目标用户：20-35岁女性
- 上传产品图片（3-5张）

**步骤2：生成文案**
- 点击"生成小红书文案"按钮
- 系统自动生成爆款文案（流式输出）
- 文案自动去除特殊符号

**步骤3：发布到小红书**
- 点击"复制文案"按钮
- 点击"下载所有图片"按钮
- 点击"发布到小红书"按钮
- 在小红书中粘贴文案并上传图片

## 使用技巧

### 我有产品功能
1. **产品信息要详细**：填写完整的产品信息有助于生成更精准的文案
2. **图片要高质量**：清晰的产品图片更能吸引用户
3. **多次生成对比**：可以多次生成文案，选择最佳版本
4. **手动优化文案**：生成后可以根据需要调整文案

### 帮我选品功能
1. **关键词要具体**：使用具体的产品名、品牌名、品类名
2. **分析爆款特点**：观察爆款笔记的标题、内容、互动数据
3. **学习创作思路**：参考爆款笔记的创作方式和表达方式
4. **找准目标用户**：分析爆款笔记的目标用户群体
5. **挖掘内容方向**：根据爆款笔记找到适合自己的创作方向

## 常见问题

### Q1：为什么搜索不到结果？
A：可能是关键词太宽泛或太冷门，建议：
- 使用更具体的关键词
- 尝试不同的关键词组合
- 搜索热门品类或品牌

### Q2：如何判断一个笔记是否值得参考？
A：可以从以下几个维度判断：
- 点赞数和收藏数的比例（收藏数高说明内容有价值）
- 评论数（评论多说明互动性强）
- 标题和内容的吸引力
- 与自己产品的相关性

### Q3：如何利用爆款笔记进行创作？
A：建议：
- 分析爆款笔记的标题结构
- 学习爆款笔记的内容框架
- 参考爆款笔记的表达方式
- 结合自己的产品特点进行创新
- 不要直接抄袭，要有自己的特色

### Q4：生成的文案可以直接使用吗？
A：建议：
- 生成后先检查文案是否符合预期
- 根据需要进行手动调整
- 确保文案符合小红书社区规范
- 添加个人特色和真实感受

## 最佳实践

### 完整的创作流程

1. **选品阶段**
   - 使用"帮我选品"功能搜索目标赛道
   - 分析爆款笔记的共同特点
   - 确定产品定位和目标用户

2. **准备阶段**
   - 拍摄高质量的产品图片
   - 整理产品的核心卖点
   - 明确目标用户的需求

3. **创作阶段**
   - 使用"我有产品"功能生成文案
   - 参考爆款笔记的创作思路
   - 手动优化文案，增加个人特色

4. **发布阶段**
   - 复制文案到小红书
   - 上传产品图片
   - 添加相关话题标签
   - 选择合适的发布时间

5. **优化阶段**
   - 观察笔记的数据表现
   - 分析用户的评论反馈
   - 持续优化创作方向
