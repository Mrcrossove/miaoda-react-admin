# 电商拿货平台功能

## 功能概述

在"帮我选品"页面新增电商拿货平台快捷入口，用户输入关键词后，除了可以查看小红书爆款笔记，还可以一键跳转到各大电商平台查看对应商品货源。

## 功能特点

### 1. 支持的电商平台

| 平台 | 图标 | 特点 | 适用场景 |
|------|------|------|----------|
| **1688** | 🏭 | 批发拿货 | 大批量采购，价格最优，适合有稳定销量的商家 |
| **淘宝** | 🛒 | 零售选品 | 小批量测试，品类丰富，适合新品测试 |
| **拼多多** | 🎁 | 低价货源 | 价格最低，适合价格敏感型产品 |
| **京东** | 🐶 | 品质保障 | 品质可靠，物流快速，适合高端产品 |

### 2. 使用流程

```
1. 输入关键词（如"口红"）
   ↓
2. 显示电商拿货平台卡片
   ↓
3. 点击任意平台按钮
   ↓
4. 在新窗口打开对应平台的搜索结果页
```

### 3. 界面设计

- **渐变背景**：使用primary/secondary渐变，与应用主题一致
- **平台图标**：每个平台有独特的emoji图标和渐变色
- **悬停效果**：鼠标悬停时卡片放大、显示外链图标
- **响应式布局**：移动端2列，桌面端4列
- **拿货建议**：底部提供各平台的使用建议

## 技术实现

### 1. 平台配置

```typescript
const E_COMMERCE_PLATFORMS = [
  {
    name: '1688',
    icon: '🏭',
    color: 'from-orange-500 to-red-500',
    url: (keyword: string) => `https://s.1688.com/selloffer/offer_search.htm?keywords=${encodeURIComponent(keyword)}`,
    description: '批发拿货',
  },
  // ... 其他平台
];
```

### 2. URL构建

每个平台的搜索URL格式不同，使用函数动态生成：

- **1688**: `https://s.1688.com/selloffer/offer_search.htm?keywords={关键词}`
- **淘宝**: `https://s.taobao.com/search?q={关键词}`
- **拼多多**: `https://mobile.yangkeduo.com/search_result.html?search_key={关键词}`
- **京东**: `https://search.jd.com/Search?keyword={关键词}`

### 3. 关键词编码

使用`encodeURIComponent()`对关键词进行URL编码，确保特殊字符正确传递。

### 4. 新窗口打开

使用`window.open(url, '_blank')`在新标签页打开，不影响当前页面。

## 使用场景

### 场景1：新品选品

1. 在小红书搜索"口红"，查看爆款笔记
2. 了解热门款式、颜色、品牌
3. 点击"1688"查看批发价格
4. 点击"淘宝"对比零售价格
5. 决定进货策略

### 场景2：价格对比

1. 输入"手机壳"
2. 同时打开多个平台
3. 对比不同平台的价格
4. 选择性价比最高的货源

### 场景3：品质验证

1. 搜索"化妆品"
2. 先在京东查看正品价格
3. 再在1688查看批发价格
4. 评估利润空间

## 注意事项

### 1. 跨域问题

直接跳转到外部网站，不存在跨域问题。

### 2. 平台限制

部分平台可能有反爬虫机制，建议用户正常浏览。

### 3. URL有效性

各平台的搜索URL可能会更新，需要定期检查和更新。

### 4. 移动端兼容性

拼多多使用移动端URL，在桌面端访问会自动跳转。
