# 自媒体创作 - 移动端网页版

## 项目说明

这是一个专为移动端浏览器优化的网页应用，不需要打包成APP，直接在手机浏览器中使用即可。

## 技术栈

- React + TypeScript
- Tailwind CSS + shadcn/ui
- Supabase (后端数据库)
- Vite (构建工具)

## 移动端优化

### 1. 响应式设计
- 所有页面都针对移动端屏幕尺寸优化
- 使用Tailwind CSS的响应式类进行适配
- 支持竖屏和横屏显示

### 2. 触摸交互
- 所有按钮和交互元素都符合移动端触摸标准（最小44x44px）
- 优化滑动、点击等手势操作
- 支持移动端浏览器的原生手势

### 3. 性能优化
- 图片懒加载
- 代码分割
- 资源压缩
- 快速加载体验

### 4. PWA特性
- 可添加到主屏幕
- 离线缓存（部分功能）
- 全屏显示模式
- 状态栏主题色适配

## 使用方式

### 开发环境
```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

### 生产部署
应用已自动部署到云端，用户可以直接通过浏览器访问：
- 在手机浏览器中打开应用URL
- 点击浏览器菜单中的"添加到主屏幕"
- 即可像APP一样使用

## 功能模块

1. **我有产品** - 产品展示与文案生成
2. **帮我选品** - 小红书爆款笔记搜索
3. **分析同行** - 同行内容分析
4. **图文创作** - 图文内容二次创作
5. **图片工厂** - AI智能图片生成
6. **电商视频** - 电商视频生成
7. **认知觉醒** - AI智能体问答

## 移动端浏览器兼容性

- ✅ iOS Safari 14+
- ✅ Android Chrome 90+
- ✅ Android Firefox 90+
- ✅ 微信内置浏览器
- ✅ 其他现代移动端浏览器

## 注意事项

1. **网络连接**：部分功能需要网络连接才能使用
2. **浏览器权限**：某些功能可能需要授予浏览器权限（如相机、相册等）
3. **存储空间**：应用会使用浏览器的本地存储保存用户数据
4. **更新方式**：刷新页面即可获取最新版本
