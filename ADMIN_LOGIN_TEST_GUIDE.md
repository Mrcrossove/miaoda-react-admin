# 管理员登录测试指南

## 测试目的
验证管理员登录功能是否正常工作，确保可以成功登录并创建用户账号。

## 测试环境
- 应用类型：移动端网页
- 浏览器：Chrome、Safari、Firefox等现代浏览器
- 修复时间：2026-01-08

## 测试步骤

### 1. 访问管理员登录页面
- 在浏览器中打开应用
- 访问路径：`/admin-login`
- 应该看到"管理员登录"页面，带有紫色渐变背景

### 2. 输入登录信息
- **用户名**: `admin`
- **密码**: `admin123`
- 点击"登录"按钮

### 3. 验证登录成功
- ✅ 应该显示"登录成功"提示
- ✅ 自动跳转到管理员后台（`/admin/accounts`）
- ✅ 页面显示"账号管理"界面
- ✅ 可以看到"创建新账号"和"批量创建账号"按钮

### 4. 创建单个用户账号
1. 点击"创建新账号"按钮
2. 填写用户信息：
   - 用户名：例如 `user001`
   - 密码：可以手动输入或点击"生成密码"
   - 显示名称：例如 `测试用户001`
   - 备注：可选
3. 点击"创建账号"按钮
4. ✅ 应该显示"创建成功"提示
5. ✅ 账号列表中应该出现新创建的账号

### 5. 批量创建用户账号
1. 点击"批量创建账号"按钮
2. 输入要创建的数量：例如 `5`
3. 点击"开始创建"按钮
4. ✅ 应该显示创建进度
5. ✅ 创建完成后显示成功提示
6. ✅ 可以导出账号列表

### 6. 查看账号列表
- ✅ 显示所有已创建的账号
- ✅ 包含用户名、显示名称、创建时间、过期时间等信息
- ✅ 可以搜索和筛选账号

### 7. 编辑账号
1. 点击某个账号的"编辑"按钮
2. 修改显示名称或备注
3. 点击"保存"
4. ✅ 应该显示"保存成功"提示

### 8. 删除账号
1. 点击某个账号的"删除"按钮
2. 确认删除操作
3. ✅ 应该显示"删除成功"提示
4. ✅ 账号从列表中消失

## 常见问题排查

### 问题1：登录时显示"账号不存在或已被禁用"
**状态**: 已修复（2026-01-08）

**原因**: admin_users表的RLS策略阻止了登录

**解决方案**: 已禁用admin_users表的RLS

**验证**: 
```sql
-- 检查RLS状态
SELECT tablename, rowsecurity 
FROM pg_tables 
WHERE schemaname = 'public' AND tablename = 'admin_users';
-- 应该返回 rowsecurity = false
```

### 问题2：登录后无法创建账号
**可能原因**: 
- 用户名已存在
- 密码格式不正确
- 网络连接问题

**解决方案**:
- 使用不同的用户名
- 使用"生成密码"功能
- 检查网络连接

### 问题3：账号创建成功但无法登录
**可能原因**: 
- 账号已过期
- 账号被禁用
- 密码输入错误

**解决方案**:
- 在管理员后台延长账号有效期
- 检查账号状态
- 重置密码
